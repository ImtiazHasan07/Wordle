# Wordle Game

Welcome to the Wordle Game project, a JavaScript application built using HTML, CSS, and vanilla JS. This project recreates the popular word-guessing game in a simple, browser-based format.

## Features

* Classic Wordle-style gameplay
* Real-time feedback on guesses (correct letters and positions)
* Limited number of attempts
* Clean and responsive user interface

## Prerequisites

Before running the Wordle Game, ensure you have the following:

* A modern web browser (Chrome, Edge, Firefox, etc.)

## Installation

Follow these steps to get your Wordle Game up and running:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/imtiazhasan07/wordle.git
   cd wordle
   ```

## Usage

To launch the Wordle Game, simply open the HTML file in your browser:

```bash
open index.html
```

Or manually open `index.html` in your preferred web browser.

Start guessing words and use the feedback to find the correct answer within the allowed attempts.
